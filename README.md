# LogicMA

O LogicMA é uma plataforma inteligente de monitoramento criminal e gerenciamento de ocorrências urbanas integrada aos distritos policiais da cidade de São Luís - MA

O sistema foi desenvolvido com o objetivo de auxiliar a segurança pública através de:
* registro de denúncias;
* classificação de prioridade;
* dashboards analíticos;
* mapas interativos;
* relatórios operacionais;

A plataforma permite que cidadãos registrem ocorrências de forma simples e rápida, enquanto gestores e agentes de segurança podem visualizar estatísticas, regiões críticas e padrões criminais em tempo real


# ▶️ Como Rodar o Projeto
## Instalar dependências
npm install
## Rodar projeto
npm run dev
